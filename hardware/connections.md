This is the pin connection for the project

| Module         | Label      | ESP32 Pin   | Description         |
|----------------|------------|-------------|---------------------|
| SD Card        | VCC        | VCC         | Power               |
| SD Card        | GND        | GND         | Ground              |
| SD Card        | CS         | GPIO12      | SD Chip Select      |
| SD Card        | MOSI       | GPIO23      | SD Data In          |
| SD Card        | SCK        | GPIO18      | SD Clock            |
| SD Card        | MISO       | GPIO19      | SD Data Out         |
| TFT Display    | VCC        | VCC         | Power               |
| TFT Display    | GND        | GND         | Ground              |
| TFT Display    | CS         | GPIO17      | TFT Chip Select     |
| TFT Display    | RESET      | GPIO5       | TFT Reset           |
| TFT Display    | D/C        | GPIO16      | Data/Command        |
| TFT Display    | MOSI       | GPIO23      | TFT Data In         |
| TFT Display    | SCK        | GPIO18      | TFT Clock           |
| TFT Display    | LED        | GPIO32      | TFT Backlight       |
| TFT Display    | MISO       | GPIO19      | TFT Data Out        |
| Touch          | T_SCK      | GPIO18      | Touch Clock         |
| Touch          | T_CS       | GPIO21      | Touch Chip Select   |
| Touch          | T_DI       | GPIO23      | Touch Data In       |
| Touch          | T_DO       | GPIO19      | Touch Data Out      |
| Touch          | T_IRQ      | (unused)    | Touch Interrupt     |
